# 屏幕十字标尺 1.1.0 / Screen Cross Ruler 1.1.0

发布日期 / Release date: 2026-07-18

## 中文

1.1.0 是首个公开发布版本，在原有的厘米十字标尺、双缩放模式和多显示器支持上新增完整外观与显示控制：

- 自定义标尺颜色。
- 中文 / English 即时切换，默认中文。
- 毫米刻度开关，默认关闭。
- 坐标数字开关，默认打开。
- 中心圆透明开关，默认打开。
- 新增“锁定标尺”，锁定后所有拖动手柄穿透鼠标，防止误操作。
- 设置窗口显示版本号、作者 `wenshuishi0528` 和 CC BY 4.0 许可。
- 修复应用已在运行时再次双击没有反馈的问题，并支持把标尺找回到鼠标所在屏幕。

下载建议：普通用户选择 universal ZIP；PKG 未签名，仅用于本机测试安装。两个产物均未做 Apple Developer ID 公证。

## English

Version 1.1.0 is the first public release. It adds complete appearance and visibility controls to the centimeter-based crosshair ruler, dual resize modes, and multi-display support:

- Custom ruler color.
- Instant Chinese / English switching, with Chinese as the default.
- Optional millimeter ticks, off by default.
- Coordinate number visibility, on by default.
- Transparent center toggle, on by default.
- New ruler lock that makes every drag handle click-through.
- Version, author `wenshuishi0528`, and CC BY 4.0 details in Settings.
- Reopen handling that brings a running ruler back to the display under the cursor.

Most users should download the universal ZIP. The unsigned PKG is intended for local testing. Neither artifact is notarized with an Apple Developer ID.
